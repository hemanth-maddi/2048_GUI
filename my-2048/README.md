# React 2048 Game

A fully functional 2048 game built with **React**. Supports dynamic board sizes (3×3 to 6×6) and tracks high scores in the browser using `localStorage`.

---

## 🎮 Gameplay

The objective of 2048 is to combine tiles with the same number to reach **2048**.

- Use **Arrow Keys** (↑ ↓ ← →) to move tiles.
- Tiles slide in the chosen direction and combine when two tiles of the same value touch.
- The game ends when no moves are possible.
- Your **score** increases as tiles combine.
- The **high score** is saved in your browser.

---

## ⚡ Features

- Dynamic board size selection (3×3 → 6×6).
- Highest score tracking (persistent in browser via `localStorage`).
- Responsive board layout.
- Color-coded tiles for easy visualization.
- Restart game option.
